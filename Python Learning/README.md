# QA AI Portfolio — Python Learning

A structured Python learning series focused on Software Development Engineer in Test (SDET) concepts.

## Files

| File | Topic |
|------|-------|
| [01_Hello_python.py](01_Hello_python.py) | Hello World & print basics |
| [02_Basic.py](02_Basic.py) | Variables and basic data types |
| [04_identified.py](04_identified.py) | Identifiers & naming rules |
| [05_keywords.py](05_keywords.py) | Python reserved keywords |
| [06_Case.py](06_Case.py) | Case sensitivity in Python |
| [07_Operators.py](07_Operators.py) | Arithmetic, comparison, logical operators |
| [08_Literal.py](08_Literal.py) | All Python literal types (int, float, string, bool, None, list, tuple, dict, set, ellipsis) |

## Literal Types Covered (08_Literal.py)

- **Numeric** — integer (decimal, octal, hex, binary), float, complex
- **String** — single/double/triple quotes, raw strings `r""`, byte strings `b""`, f-strings
- **Boolean** — `True` / `False`
- **None** — singleton `None`
- **Collection** — list `[]`, tuple `()`, dict `{}`, set `{}`
- **Ellipsis** — `...` placeholder and type-hint usage

## Setup

```bash
# Requires Python 3.6+
python --version

# Run any file
```
